# Cobra Minus 0C
**Cobra** is a Python IDE created by **terni.** which aims to make learning programming more palatable to new developers, while also be welcoming to experienced engineers.

The terni. IDE suite is planned to expand to other programming languages, like C, C++, Rust, etc.

This is Cobra Minus, the free and open-source version of Cobra.

## History
Cobra was initially powered by Java and started it's journey on January 23rd, 2025. Eventually, some problems were occurring with the compiling of the software and it was decided that Cobra would move to a different language, being C++ and wxWidgets. 

This was called the 0B.2 version, with B referring to a new language being used.

C++ was used until recently, April 2nd, 2026*, when it was decided that the language was too unsafe and complex. It would take a long time debugging and fixing bugs when it would be safer and better in the long run to switch to a different language before it's too late.

Now, Cobra 0C.5 is powered by Rust and egui.

<em>*Iced was the framework used in February 17th until there were issues with the iced-aw crate.</em>

## Possible Future of Builds
It's likely that Cobra will either return to builds A and B and redo them, more likely the former, to create Cobra 0Aa.x. Or, we could see more languages being used, like Python for a build D, or even C# for a build E.
